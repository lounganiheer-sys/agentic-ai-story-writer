# Agentic AI Story Writer (Gemini API Powered)

## Overview
This project is an Agentic AI Story Generator built using Python and Gemini API.
It generates structured stories based on user prompts.

## Features
- User input prompt
- AI-powered story generation
- PDF export using ReportLab / FPDF
- Streamlit interface

## Technologies Used
- Python
- Streamlit
- ReportLab
- FPDF
- Gemini API

## How to Run

1. Install requirements:
setx Gen_API ""
pip install -r requirements.txt
python -m streamlit run story.py

2. Run the app:
   streamlit run story.py
